# 融资黑话翻译局

看穿 VC 和创始人的真实想法。

一个帮你翻译融资圈"黑话"的小工具，把那些听起来高大上的话翻译成人话。

## 功能

- **创始人模式** - 翻译创始人常说的融资话术
- **投资人模式** - 翻译投资人的委婉表达
- **可信指数** - 评估这句话的"水分"含量
- **一键分享** - 保存图片发朋友圈吐槽

## 技术栈

- React + TypeScript
- Vite
- Tailwind CSS
- DeepSeek API

## 本地运行

1. 安装依赖：
   ```bash
   npm install
   ```

2. 创建 `.env.local` 文件，配置 DeepSeek API Key：
   ```
   DEEPSEEK_API_KEY=your_api_key_here
   ```

3. 启动开发服务器：
   ```bash
   npm run dev
   ```

## 设计风格

复古报刊风 - 奶油纸色背景、黑色边框、硬阴影，像一份正经的"官方鉴定报告"。
